# 👩‍💻 Web Scraping usando Puppeteer
Web Scraping: ato de resgatar informações de páginas da WEB

Comandos inciais:
- npm init -y 
- npm install express 
- npm install puppeteer

Comando para executar o projeto:
- node app.js
#
